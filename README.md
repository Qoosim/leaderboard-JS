## 🖥️ Getting Started
To get a local version up and running on your machine, follow these simple example steps.

## 🛠️ Prerequisites
```
  Install Node.js on your machine
```

## 🕹️ Setup
```
  Clone the repository on your machine and cd into it
  run npm install
  run npm run start
```
